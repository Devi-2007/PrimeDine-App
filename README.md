**PrimeDine** is a Python-based console food ordering application that allows users to login with phone/email OTP, browse the menu, add items to the cart, and place orders.  

---

## Features

- Login with **Phone** or **Email** using OTP verification  
- Browse menu categories: Veg, Non-Veg, Snacks, Desserts, Drinks  
- Add items to **cart** with quantity-based pricing  
- View cart and **place order**  
- Logout functionality
